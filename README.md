# 斗罗大陆 Soul Land Mod 中文汉化补丁

> Minecraft Forge 1.20.1 · soulland-1.0.0.jar

## 一键使用

把 `soulland_translate.py` 和 `soulland_translations.json` 放在**同一目录**下，然后：

```bash
python3 soulland_translate.py soulland-1.0.0.jar
```

脚本会自动：
1. 备份原 jar 为 `.bak`
2. 注入 `zh_cn.json` 语言文件（89 条）
3. 翻译 41 个成就文件
4. 替换 class 文件中的硬编码英文 → 中文（2463 条翻译，~2252 处命中）

完成后重启 Minecraft 即可。

## 翻译原则

- **术语忠于原著**：Soul Ring → 魂环、Martial Soul → 武魂、Spirit Beast → 魂兽
- **人名/地名按小说**：Tang San → 唐三、Shrek Academy → 史莱克学院
- **保持格式不变**：§ 颜色代码、%s/%.0f 占位符原样保留

## 文件说明

| 文件 | 说明 |
|------|------|
| `soulland_translate.py` | 主脚本，负责注入和替换 |
| `soulland_translations.json` | 英→中翻译字典（2463 条） |

两个文件必须在同一目录。

## 技术原理

脚本通过修改 Java class 文件的**常量池 UTF-8 条目**（tag=1）来替换硬编码字符串。对于每条翻译：

1. 在 class 二进制数据中搜索原文 UTF-8 字节
2. 验证前 2 字节为合法的长度头
3. 替换内容并更新长度头

**安全边界**：
- ✅ 只改 UTF-8 常量池条目
- ❌ 绝不修改数字字节码（sipush）— 会崩溃
- ❌ 绝不翻译 UPPER_CASE 注册名（如 `BODY_TEMPERING_PILL`）— 物品查找会失败
- ❌ 绝不翻译 camelCase 代码标识符 — 不是显示文本

## 已知限制

- 部分动态生成的文本（由代码拼接的字符串）可能仍显示英文
- GUI 面板高度未修改，建议调整 GUI 缩放比例避免溢出
- 仅适用于 `soulland-1.0.0.jar`（Forge 1.20.1）
